# Dockit
## - What is this?
Fuzzy current location to git pull with all submodules or launch, close and exec container through docker-compose.

## - How to install
```bash
pip install Dockit
```

## - How to use

### CLI

```bash
dockit -p
```

```bash
dockit -c
```
